# CSS Animations
A page to test CSS animations during my training at Interface3 - Brussels.

A bouncy logo, and a smooth animation of nav links on hover.
A small scooter crosses the page from left to right and back again.


## Technologies Used

* HTML5
* CSS3
* VS Code
